IBD scRNA-seq and Spatial Transcriptomics Metabolism Analysis

Reproducible inventory, baseline summaries, and planned Harreman workflow

Author

Matthew Chung

Published

April 22, 2026

1 Introduction

Stricturing Crohn’s disease involves inflammation, tissue remodeling, fibrosis, and altered epithelial and stromal biology. This report tracks a reproducible analysis of paired single-cell RNA-seq and Visium spatial transcriptomics data for stricturing Crohn’s disease, with an emphasis on metabolism-aware spatial inference.

The initial goals are:

  • establish a reproducible Python workflow using uv;
  • inventory the single-cell and spatial data objects;
  • generate baseline scRNA-seq and spatial transcriptomics summaries;
  • build quality-control outputs that can be reused in downstream reporting;
  • prepare for metabolic pathway scoring and Harreman-based spatial metabolic crosstalk inference (Etxezarreta Arrastoa et al. 2025; Yosef Lab 2026).

The dataset was obtained from Zenodo record 10.5281/zenodo.14509802 (Kong et al. 2024). Analysis uses AnnData-compatible Python tooling, including Scanpy (Wolf, Angerer, and Theis 2018) and Squidpy (Palla et al. 2022).

2.2 Software Environment

The project is managed with uv. Direct pip install is not used.

The current analysis environment was initialized with:

uv init --name ibd-scrna-strna-metabolism --python 3.12
uv add scanpy squidpy decoupler harreman openpyxl pyarrow
uv add 'scanpy<1.12'

Harreman 0.1.3 required a Scanpy version below 1.12 because it imports a private Scanpy helper that is not available in Scanpy 1.12. The project is therefore pinned to scanpy<1.12.

The ambient Anaconda environment had a broken anndata/h5py binary stack, so all Python analysis should be run through the project environment:

uv run python scripts/00_inventory.py
uv run python scripts/01_baseline_overview.py

2.3 Analysis Pipeline

Two scripts have been implemented so far.

scripts/00_inventory.py inventories the data directory, raw spatial archives, metadata tables, and AnnData objects using backed AnnData reads where possible. It writes outputs to output/inventory/.

scripts/01_baseline_overview.py creates first-pass scRNA-seq and spatial metadata summaries. It writes outputs to output/baseline_overview/.

scripts/02_spatial_qc.py creates per-slide spatial coordinate plots and low-resolution tissue-image overlays using the raw Visium archive contents. For each slide, the script reads the archive position list, scale factors, and tissue image, then verifies that AnnData spot barcodes are present in the raw archive.

scripts/03_metabolism_scores.py computes first-pass spatial scores for curated metabolism and fibrosis-relevant gene sets. Scores are calculated as the mean expression of matched genes in the spatial AnnData X matrix, followed by global z-scoring across spots. These scores are intended as sanity checks and exploratory spatial summaries before Harreman, not as formal metabolic flux estimates.

scripts/04_harreman_pilot.py runs a Harreman transporter communication pilot on a paired adjacent and fibrotic slide subset from patient 110201. It extracts the human extracellular transporter database, constructs a spatial nearest-neighbor graph, filters sparse transporter genes, computes transporter gene pairs, and runs Harreman’s parametric communication test on CPU.

scripts/05_harreman_all_slides.py scales the same Harreman workflow to all paired adjacent and fibrotic slides. It writes per-sample cached outputs, so interrupted or repeated runs can resume without recomputing completed slides.

scripts/06_spatial_deconvolution.py performs first-pass spatial deconvolution using the provided CIBERSORTx Gut Atlas IBD signature matrix. It fits non-negative least-squares coefficients for each Visium spot using 2,000 shared high-variance signature genes, then normalizes coefficients into cell-type proportions.

scripts/07_harreman_deconv_interpretation.py connects Harreman metabolite shifts to deconvolved cell-type proportions and marker-gene validation. It computes slide-level and paired-difference correlations, exports top evidence tables, and checks expected marker-set correlations for major deconvolved cell types.

scripts/08_transporter_spatial_validation.py validates focused Harreman metabolites by tracing them back to their transporter genes in the Harreman database. It summarizes transporter expression by slide, performs paired fibrotic-minus-adjacent expression comparisons, correlates transporter expression with Harreman metabolite z-scores, and exports all-slide spatial expression maps for representative transporter genes.

scripts/09_spatial_region_validation.py moves from slide-level validation to spot-level spatial-region validation. It defines high-signal regions within each slide from deconvolution and pathway scores, extracts focused transporter-gene expression from the spatial AnnData object, and tests whether transporter genes are enriched in epithelial, bile-acid, stromal-fibrotic, and inflammatory regions.

scripts/10_scrna_metabolism_transporters.py summarizes metabolism gene-set scores and Harreman-relevant transporter genes in the annotated scRNA-seq object. Because the single-cell object is large, the script reads the H5AD CSR matrix directly in row chunks and computes only selected pathway and transporter features. It then summarizes those features by cell type, disease status, fraction, and donor.

scripts/11_integrated_evidence_table.py joins the major output layers into a compact evidence table for priority metabolites. The table links Harreman direction, transporter-gene support, spatial-region enrichment, deconvolved cell-type associations, and scRNA-seq supporting cell types.

scripts/12_patient_paired_models.py runs patient-paired statistical models on the integrated outputs. For each feature, the script treats the patient-level fibrotic-minus-adjacent difference as the unit of analysis, then computes mean difference, bootstrap confidence interval, paired effect size, paired t-test, Wilcoxon signed-rank test, direction consistency, and Benjamini-Hochberg FDR within each analysis layer.

scripts/13_region_sensitivity.py evaluates whether priority spatial-region transporter results depend on the chosen high-region threshold. It reuses the continuous region scores and transporter expression table, then repeats region-level summaries using the top 10%, 20%, and 30% of spots per slide as high-signal regions.

scripts/14_deconvolution_sensitivity.py compares CIBERSORTx NNLS deconvolution with independent marker-gene score summaries. It evaluates expected marker-to-cell-type pairs at slide level and patient-paired difference level.

scripts/15_consolidated_figures.py creates report-ready summary figures from existing outputs. It does not rerun heavy analyses; it consolidates the strongest Harreman, transporter, region-sensitivity, deconvolution-sensitivity, and scRNA-seq evidence into a compact panel and interpretation schematic.

scripts/16_healthy_control_comparisons.py adds explicit healthy-control comparisons before manuscript-style interpretation. In scRNA-seq, status == "H" is used as the healthy reference. In spatial transcriptomics, the two Adjacent (Disease Control) slides are used as a descriptive disease-control reference.

The current workflow avoids loading the full expression matrices into memory during inventory and metadata summarization.

4.5 Metabolism And Fibrosis Scores

  • retinoid/vitamin A biology;
  • ECM remodeling;
  • inflammation;

All requested genes were present in the spatial AnnData object for each gene set. Scores were summarized per slide and by disease label. At the disease-label level, fibrotic slides had the highest mean scores for inflammation and ECM remodeling among the tested programs. 4.8 Spatial Deconvolution

The provided CIBERSORTx Gut Atlas IBD signature matrix was used as a direct reference for first-pass Visium spot deconvolution. The matrix contains 42 cell types. Of 7,458 signature genes, 7,350 overlapped the spatial AnnData gene symbols. The initial NNLS model used the 2,000 shared genes with the highest variance across signature cell types.

Metric Value
Spots deconvolved 58,815
Signature cell types 42
Signature genes 7,458
Signature genes overlapping spatial data 7,350
Selected genes used in NNLS 2,000
Median relative residual 0.821
Mean relative residual 0.825

The high residual indicates that this transparent NNLS deconvolution is useful as an initial composition estimate, but should be interpreted cautiously. Potential reasons include platform differences between the CIBERSORTx scRNA-seq reference and Visium profiles, broad spot mixtures, scale differences between signature values and spatial expression values, and the use of a deliberately simple constrained regression model.

Mean paired fibrotic-minus-adjacent differences suggested higher B/plasma signal in fibrotic slides and lower absorptive, secretory, myofibroblast, smooth muscle, fibroblast, lymphatic endothelial, microfold, vascular endothelial, and pericyte signals on average. Some of these directions are biologically plausible, while others need scrutiny against histology, marker expression, and fit diagnostics before interpretation.

4.19 Manuscript-Style Results Draft

4.19.1 Spatial metabolism inference identifies loss of epithelial bile-acid and urea transporter programs in fibrotic tissue

Across paired adjacent and fibrotic spatial transcriptomics slides, Harreman prioritized a set of epithelial-associated metabolites that were lower in fibrotic tissue. Urea, bile acid, and cholic acid showed negative mean fibrotic-minus-adjacent Harreman z-score differences. These signals were supported by transporter-level evidence. SLC5A1, which was linked to the urea signal in the Harreman transporter database, correlated strongly with slide-level Harreman urea scores and was enriched in epithelial bile and absorptive regions. SLC10A2, SLC51A, and SLC51B supported the cholic-acid and bile-acid signals and were likewise enriched in epithelial regions.

The epithelial interpretation was consistent across analysis layers. Harreman bile-acid, cholic-acid, and urea scores correlated with absorptive and microfold deconvolution estimates. Independent marker-gene sensitivity supported the absorptive epithelial direction: absorptive marker scores and NNLS absorptive deconvolution both decreased in fibrotic-minus-adjacent comparisons. The scRNA-seq reference also supported this assignment, with bile-acid transport and epithelial bile/urea transporter scores highest in enterocyte states.

Healthy-control comparisons strengthened this epithelial axis. In scRNA-seq, fibrotic Enterocyte-FABP-hi and Enterocyte-ANPEP-hi states had lower bile-acid transport scores than healthy-control cells. In spatial transcriptomics, the two disease-control adjacent slides had higher epithelial bile/urea transporter signals than Crohn’s adjacent and fibrotic slides. Fibrotic slides showed lower epithelial-bile-region SLC5A1, epithelial-bile-region SLC10A2, bile-acid transport score, slide-level SLC5A1, epithelial-bile-region SLC51A, and epithelial-bile-region SLC51B than disease-control slides. Because the spatial disease-control group contains only two slides, these comparisons should be interpreted as descriptive rather than confirmatory.

4.19.2 Fibrotic tissue shows a candidate amino-acid and organic-anion transporter program associated with inflammatory and stromal regions

Harreman also identified fibrotic-shifted metabolites, including L-histidine, L-serine, L-cysteine, and citric acid. These signals were supported by transporter genes including SLC38A2, SLC15A3, SLC1A5, SLC7A5, and SLCO2B1. Region-level analysis showed that SLC38A2, SLC15A3, and SLCO2B1 were higher in stromal-fibrotic or immune-inflammatory regions than in epithelial bile regions. The direction of these region-level transporter signals was stable across high-region definitions using the top 10%, 20%, and 30% of spots per slide.

The scRNA-seq reference assigned this transporter program primarily to macrophage-like and stromal-associated populations. The fibrotic amino-acid transporter panel was highest in M2-like macrophage, CD63+CD81+ macrophage, monocyte, and inflammatory fibroblast populations. ECM remodeling scores were highest in trophocyte, telocyte, myofibroblast, pericyte, tissue fibroblast, and inflammatory fibroblast populations. Compared with healthy-control cells, fibrotic inflammatory fibroblasts and myofibroblasts showed higher ECM remodeling scores.

This fibrotic transporter axis should be interpreted more cautiously than the epithelial axis. Patient-paired models did not identify FDR-supported features at 0.10 across the tested analysis layers. In addition, paired marker-score and NNLS deconvolution directions disagreed for fibroblast, myofibroblast, and vascular endothelial signals. For this reason, stromal and fibrotic interpretations should rely on convergent transporter, region-score, marker-score, and scRNA-seq evidence rather than NNLS deconvolution alone.

4.19.3 Patient-level modeling supports effect-size prioritization but not confirmatory differential claims

Patient-paired statistical modeling treated each fibrotic-minus-adjacent difference as one observation. Across 355 modeled features and 45 priority features, no feature passed both paired t-test FDR < 0.10 and Wilcoxon FDR < 0.10. The strongest priority effect-size summaries nevertheless matched the cross-layer interpretation: stromal-fibrotic-region SLC38A2 was higher in fibrotic tissue, epithelial-region SLC10A2 was lower in fibrotic tissue, stromal-fibrotic-region SLC15A3 was higher in fibrotic tissue, slide-level SLC10A2 was lower in fibrotic tissue, slide-level SLCO2B1 and SLC15A3 were higher in fibrotic tissue, Harreman citric acid was higher in fibrotic tissue, and spatial bile-acid transport was lower in fibrotic tissue.

Together, the current results support two metabolism-associated spatial programs. The strongest is an epithelial bile-acid and urea transporter program that is reduced in fibrotic Crohn’s tissue and lower than the limited disease-control spatial reference. The second is a candidate fibrotic amino-acid and organic-anion transporter program associated with macrophage-like, inflammatory, and stromal-fibrotic contexts. The latter remains hypothesis-generating and requires stronger validation before being interpreted as a confirmed disease-associated metabolic program.

5 Next Analyses

The next step is interpretation and refinement rather than another broad method run. Priority analyses are:

  • compare Harreman results with the simpler gene-set scores;
  • compare Harreman results with the CIBERSORTx signature-matrix deconvolution;
  • refine deconvolution using marker-gene diagnostics, alternative gene selection, or a heavier method if needed;
  • convert the consolidated figures and evidence tables into manuscript-style results text.

The immediate next refinement should be manuscript-style results text and figure captions. The analyses now identify a robust epithelial bile-acid and urea transporter axis that is also lower relative to spatial disease-control slides, plus a more cautious fibrotic amino-acid and organic-anion transporter axis that needs stronger validation before being treated as confirmatory.

6 Limitations

Current results include inventory, metadata-level summaries, spatial coordinate/image QC, exploratory metabolism/fibrosis gene-set scores, one Harreman pilot, an all-paired-slide Harreman transporter communication run, first-pass CIBERSORTx signature-matrix deconvolution, an initial Harreman-cell-type interpretation layer, focused transporter-gene spatial validation, region-level validation, direct scRNA-seq cell-type metabolism and transporter summaries, an integrated metabolite evidence table, patient-paired statistical modeling, region-threshold sensitivity analysis, deconvolution marker sensitivity analysis, healthy-control comparisons, and consolidated summary figures. No cell-type-aware Harreman model has been generated yet. The patient-paired models did not identify FDR-supported features at 0.10, so the current effect-size results remain hypothesis-generating. Transporter expression supports several Harreman signals, but expression is not a direct measure of metabolite abundance, metabolite exchange, or flux. Spot-level correlation p-values should also be interpreted cautiously because Visium spots are spatially autocorrelated. Spatial healthy-control comparisons are descriptive because only two disease-control adjacent slides are available. Stromal cell-type direction should be interpreted cautiously because paired marker-score and NNLS deconvolution directions disagree for fibroblast, myofibroblast, and vascular endothelial signals. The current Harreman and deconvolution results should be treated as prioritization output rather than final biological conclusions.
